Description

WolfPACS's Protocol Data Value (PDV) Framgment Item

If bit 0 is set to 1, the following fragment shall contain Message Command information. If bit 0 is set to 0, the following fragment shall contain Message Data Set information.

If bit 1 is set to 1, the following fragment shall contain the last fragment. If bit 1 is set to 0, the following fragment does not contain the last fragment.

0 0 0 0 0 0 L? C?
