WolfPACS A Open Source PACS solution written in Erlang.

WolfPACS is an DICOM load balancer written in Erlang.

With the advent of heavy ML/AI solutions in Radiology, there is growing need to split the workload across multiple workers. WolfPACS acts as a load balancer, sending DICOM series to the correct worker.
